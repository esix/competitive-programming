# X Cubic [⬀](https://judge.u-aizu.ac.jp/onlinejudge/description.jsp?id=ITP1_1_B)

Write a program which calculates the cube of a given integer `x`.

## Input
An integer `x` is given in a line.

## Output
Print the cube of `x` in a line.

# Constraints
- `1 ≤ x ≤ 100`

## Sample Input 1
```
2
```

## Sample Output 1
```
8
```

## Sample Input 2
```
3
```

## Sample Output 2
```
27
```
