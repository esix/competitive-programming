# Perimeter of rectangle [⬀](https://www.e-olymp.com/en/contests/9508/problems/83309)

Find the perimeter of rectangle.

## Input

The integer sides of rectangle a and b (1 ≤ a, b ≤ 1000).

## Output

Print the perimeter of rectangle.

## Input example #1
```
3 4
```

## Output example #1
```
14
```
