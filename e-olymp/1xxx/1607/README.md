# [The number in reverse order](https://www.e-olymp.com/en/problems/1607)
Write a nonnegative integer n in reverse order.

## Input
One nonnegative 64-bit integer.

## Output
Print the number in reverse order.

## Input example #1
```
1234
```

## Output example #1
```
4321
```

## Input example #2
```
100
```

## Output example #2
```
001
```
