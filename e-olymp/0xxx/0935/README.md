Decompose a given 3-digit number to digits.

## Input
One three digit integer (positive or negative).

## Output

## Input example #1
```
198
```

## Output example #1
```
1
9
8
```

## Input example #2
```
-232
```

## Output example #2
```
2
3
2
```
