# Sum of squares [⬀](https://www.e-olymp.com/en/problems/4725)
Given positive integer n. Find the sum 1² + 2² + 3² + ... + n².

## Input
One positive integer n.

## Output
Print the found sum.

## Input example #1
```
1
```

## Output example #1
```
1
```

## Input example #2
```
2
```

## Output example #2
```
5
```
