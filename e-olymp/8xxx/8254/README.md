# Sum of squares from 1 to n [⬀](https://www.e-olymp.com/en/problems/8254)

Given positive integer `n`. Find the sum of squares `S = 12 + 22 + ... + n²`.

## Input

One positive integer `n` (`n ≤ 10⁶`).

## Output

Print the value of the sum of squares `S`.

## Input example #1
```
3
```

## Output example #1
```
14
```

## Input example #2
```
5
```

## Output example #2
```
55
```
