# Two Strings [⬀](https://www.hackerrank.com/challenges/two-strings)

Вам даны две строки `A` и `B`. Определите, 
существует ли непустая строка, которая 
встречается как подстрока и в `A`, и в `B`.

## Input Format

В первой строке записано целое число `T` - 
количество тестов. Далее следуют сами тесты. 
Каждый тест описывается двумя строками, в 
первой из которых записана строка `A`, а во 
второй строка `B`.

## Output Format

Для каждого теста выведите ответ: строку `YES`, 
либо строку `NO`.

## Sample Input
```
2
hello
world
hi
world
```

## Sample Output
```
YES
NO
```

## Ограничения

- Все строки состоят из строчных символов латинского алфавита.
- `1 ≤ T ≤ 10`
- `1 ≤ |A|,|B| ≤ 10⁵`
